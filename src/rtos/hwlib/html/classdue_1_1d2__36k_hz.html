Detailed Description

36kHz output on pin chip PB25 = Arduino D2

This class provides a 36 kHz output on chip pin PB25 (Arduino pin D2) that can be enabled or disabled by calling set( 1 ) resp. set( 0 ).

enable or disable the 36 kHz output

Calling set( 1 ) enables the 36 kHz output, calling set( 0 ) disables the output and makes the output low.
